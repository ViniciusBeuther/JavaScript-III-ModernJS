Tipos de Export

    Modulos - Formas

  • Export Nomeado Inline: Porque precisa passar o nome exato dele, export vai na mesma linha. Numero ilimitado
    export function name(name){ codeblock }
    import { functionName } from "./filepath.js"

  • Default export: Podemos exportar com um nome qualquer, limitado a 1 export default funct... por arquivo (pode ser até funcao anonima)
    export default name(params){ codeblock }
    import nomeQualquer from "./filepath.js"

  • Export NonInLine é quando podemos agrupar o export, podemos usar export { function1, function2, ... }, podemos misturar default export e export
